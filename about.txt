# DataInsight AI - Application Overview

## 1. Executive Summary
DataInsight AI is an intelligent, browser-based analytics assistant that transforms raw data files (CSV, Excel) into actionable insights. By leveraging Google's Gemini 2.5 AI model, it allows users to query data using natural language, perform complex statistical analysis, and generate interactive visualizations instantly. It bridges the gap between non-technical users and advanced data analytics operations.

## 3. How to Use DataInsight AI

### Step 1: Upload Data
- Open the application.
- Drag and drop your CSV or Excel file into the "Data Source" area on the left sidebar.
- Wait a moment for the AI to parse, clean, and summarize your data.

### Step 2: Explore & Visualize
- **Read the Summary**: The AI will provide an initial overview and key stats.
- **Ask Questions**: Type in the chat box.
  - *Simple*: "Show total sales by city."
  - *Complex*: "Calculate the average profit margin per category and show the top 5."
- **Use Suggestions**: Click on the AI-generated "suggested questions" chips to get started quickly.

### Step 3: Transform & Clean (Optional)
- If your data has noise, tell the AI to clean it.
  - Example: "Remove all rows where the Status is 'Refunded'."
- Create new metrics.
  - Example: "Create a column 'Net_Price' by subtracting 'Discount' from 'Price'."
- All subsequent analysis will be performed on this new, clean version of your data.

### Step 4: Filter & Drill Down
- **Visual Filter**: If you see an interesting outlier in a chart (e.g., a spike in Sales for "Q4"), double-click that bar. The app will filter the *entire* dataset to just Q4.
- **Sidebar Filter**: You can also manually add filters using the "Data Tools" tab in the sidebar.
- Ask follow-up questions on this filtered subset.

### Step 5: Export Results
- Once you have the insights you need, use the **Export** buttons in the sidebar to download the modified dataset.
- Click the **Export PNG** button on any specific chart to save the visualization.

## 4. Tech Stack
- **Core AI**: Google Gemini 2.5 Flash (via `@google/genai` SDK).
- **Frontend**: Vanilla JavaScript (ES Modules), HTML5, CSS3.
- **Design System**: Neobrutalism (High contrast, bold typography, "Google Palette").
- **Data Engine**: `PapaParse` (CSV), `SheetJS` (Excel).
- **Visualization**: `Chart.js` with `chartjs-plugin-zoom`.

## 5. Value Proposition
- **Democratization of Data**: Empowers non-technical teams (Sales, HR, Marketing) to answer ad-hoc questions without SQL knowledge.
- **Privacy First**: All data processing happens client-side in the browser. Raw data is never stored on external servers.
- **Speed**: Reduces time-to-insight from days to seconds.
- **Zero Setup**: Works immediately in any modern web browser.
